# Context: 占城大师 MVP 原型

## Glossary

| Term | Definition |
|------|------------|
| **六边形地图** | 5×7 的平顶六边形网格，使用 axial coordinate (q, r) 表示位置 |
| **大本营** | 双方初始的已翻开格子，位于地图两端，被摧毁即判负 |
| **迷雾地块** | 初始未翻开的格子，点击消耗金币后揭示内容 |
| **翻牌** | 消耗 10 金币揭示迷雾地块内容的行为 |
| **金矿** | 翻出后给予 50 金启动资金，并每秒产出 5 金的地块 |
| **空地** | 可建造兵营的地块 |
| **野怪** | 翻出后需战斗才能占领的地块 |
| **特殊事件** | 占 10% 权重的随机事件地块 |
| **兵营** | 消耗 40 金建造在空地上，每 3 秒自动产出一个随机兵种单位 |
| **兵种** | 步兵（HP220/ATK15）、骑兵（HP180/ATK18）、弓箭手（HP120/ATK20） |
| **克制链** | 步兵 > 骑兵 > 弓箭手 > 步兵，克制方伤害 ×1.5，被克方 ×0.5 |
| **占领** | 己方单位进入中立/敌方格子后，格子归属变色 |
| **推进** | 单位沿最短路径向敌方大本营方向移动，占领相邻格子 |
| **交战** | 相邻格子的敌对单位自动互相攻击，频率 1 次/秒 |
| **AI 行为树** | 固定逻辑：每 2 秒翻牌 → 每 5 秒检查建兵营 → 兵营自动出兵 |
| **MuleRun** | 零依赖单文件 HTML 交付格式，所有资源内联 |

## Decisions

- 平台：MuleRun 单文件 HTML，浏览器直接运行
- 对战模式：单人对 AI（PvE）
- 截止日：2026-06-22
- 出兵方式：免费自动产出，玩家不消耗额外金币
- 兵种获取：翻牌随机出
- AI 难度：固定行为（原型阶段）
