# Context: 占城大师 (Hex Takeover)

> 本文件是**领域词汇表**，记录项目内每个术语的精确含义。
> 不记录实现细节、数值常量或开发流程——那些属于 AGENTS.md 和 PRD。
> 当讨论中出现术语歧义时，以本文件为准。

## Glossary

### 地图与空间

| Term | Definition |
|------|------------|
| **六边形地图** | 9 列 × 12 行 = 108 格的尖顶(pointy-top)六边形网格，odd-r offset 布局（奇数行右移半格），无湖水分隔。使用 offset coordinate (q, r) 表示位置。 |
| **领地归属** | 每个可占领格子都有"己方 / 敌方 / 中立"三种归属状态，决定谁能在该格子上行动。 |
| **初始领地** | 开局时双方各占一半地图：r=0~5 归 AI（红色），r=6~11 归玩家（绿色）。所有非 HQ 格子初始为未激活态。HQ 周围 6 格例外，直接设为可开态（中立归属，先到先得）。 |
| **主堡 (HQ)** | 双方各一个的核心建筑格子，被摧毁即判负。不可重建。 |

### 格子生命周期

| Term | Definition |
|------|------------|
| **未激活格子** | 显示领地色（绿/红）但无图标、无价格、不可操作的格子。**计入领地格数统计**。可通行、可占领。只有相邻于**己方 opened/openable 格子**时才激活为可开态。 |
| **可开格子** | 已激活等待付费开启的格子。图标与价格在激活时一次性确定，不刷新。**视觉敌我不对称**：我方可开格显示具体图标（?/头盔/盾牌/金属）+ 价格 pill；敌方可开格统一显示"恶魔脸"标记。归属为中立（owner=null），先到先得。 |
| **已开格子** | 玩家付费开启后的格子，显示具体建筑（或空）。一次性动作，不可逆。 |
| **废墟** | 建筑被摧毁后的格子状态。可通行、不可重建、**归属固定为摧毁方**（不会被经过的敌方单位再次占领）。 |

### 格子上的图标（可开态显示）

| Term | Definition |
|------|------------|
| **兵营槽 (?)** | 问号图标。开出来后进入"兵营建筑"，按价格档位走兵种品阶概率表（25 金档有"空"的可能）。 |
| **未确定兵种 (头盔)** | 头盔图标。开出来后必出一个兵种（不会空），具体哪一种按档位概率表二次随机。 |
| **箭塔槽 (盾牌)** | 盾牌图标。开出来后得到一个箭塔，具体是 50 金档还是 100 金档各 50%。 |
| **金矿槽 (金属)** | 金属图标。开出来后直接得到金矿，无二次随机。 |
| **混合池** | 可开格子图标来源的概率分布：兵营槽 55% / 头盔 25% / 箭塔槽 10% / 金矿槽 10%。 |

### 价格与概率

| Term | Definition |
|------|------------|
| **价格档位** | 25 / 50 / 100 / 250 金币四档。每个可开格子在激活时随机分配一档。 |
| **档位差异化** | 不同图标类型的价格档位分布不同：兵营槽偏 25 金（便宜赌局），头盔/箭塔槽偏 50/100 金（稳定），金矿槽偏 50 金。 |
| **二次概率表** | 兵营槽和头盔按价格档位查各自的"兵种品阶概率表"决定产出品质。档位越高，紫/金概率越高。 |
| **空 (兵营槽特例)** | 兵营槽在 25 金档位有 50% 概率"开空"——扣费但不出建筑，格子进入"已开但无产出"状态。 |

### 建筑

| Term | Definition |
|------|------------|
| **兵营建筑** | 已开的兵营槽格子。持续按该兵种的产出 CD 吐出兵种单位。是永久的产兵源。 |
| **兵种卡 (已确定)** | 已开的头盔格子。持续按该兵种的产出 CD 吐出该特定兵种。与兵营建筑行为等价，只是图标不同。 |
| **箭塔** | 已开的箭塔槽格子。自动攻击范围内的敌方单位。分 50 金档和 100 金档两档。 |
| **金矿** | 已开的金矿槽格子。被动产出金币，是经济引擎。 |
| **建筑升级** | 消耗金币提升金矿/箭塔的属性（产出速率、攻击力、射程、HP）。分三档，每档成本翻倍。兵营建筑不可升级。 |

### 兵种

| Term | Definition |
|------|------------|
| **品阶** | 绿 / 蓝 / 紫 / 金四级。品阶越高综合能力越强。**v2 不做类型克制链，纯品阶压制**。 |
| **绿阶** | 骷髅怪（近战血厚）、野狼（近战快攻）两种。 |
| **蓝阶** | 人类战士（近战+远程自动切换）、人类弓手（同上）。 |
| **紫阶** | 人类骑兵。飞行单位，可越过格子直接攻击敌方建筑。 |
| **金阶** | 龙骑兵。远程 AOE 高伤，CD 期间进入近战。 |

### 单位行为

| Term | Definition |
|------|------------|
| **逐格推进** | 非飞行单位沿六边形相邻格移动，每步走一格。 |
| **飞行越格** | 紫色骑兵专属。无视路径上的单位和格子，直接飞到目标建筑格子攻击。 |
| **防守反击寻敌** | 绿/蓝阶单位攻击离**己方阵营最近**的敌方目标，不深入敌后追击。 |
| **建筑专注寻敌** | 紫阶单位专打敌方建筑，无视路径上的敌方单位。 |
| **品阶切换寻敌** | 金阶单位远程模式优先打敌方单位，近战模式（CD 中）打最近目标。 |
| **己方穿越** | 己方单位不挡路，其他己方单位可自由穿过。 |

### 领地与推线

| Term | Definition |
|------|------------|
| **推线连锁** | 己方单位踩过敌方格子 → 格子变为己方颜色（claim）→ 若该格子相邻于己方 opened/openable 格子，则周围未激活格子被激活（成为可开格子）。未激活格子本身可通行、可占领，计入格数统计。 |
| **建筑摧毁连锁** | 敌方建筑被摧毁 → 该格子变为废墟，**归属固定为摧毁方**（不会被经过的敌方单位再次占领）。废墟可通行但不可重建。 |
| **地块数** | 某一方当前拥有的格子总数（己方为绿色格子、敌方为红色格子）。实时显示在该方头像旁，是"时间耗尽比格数"的实时呈现。 |

### 人口

| Term | Definition |
|------|------------|
| **人口** | 场上某一方当前存活的单位数量。实时显示在该方头像旁，形如"当前 / 上限"。单位死亡后名额释放。 |
| **人口上限** | 一方场上单位数量的硬上限。达到上限后该方所有产兵源（兵营建筑、兵种卡）暂停产出，直到有名额释放。是约束战斗规模的核心机制。 |

### 胜利与失败

| Term | Definition |
|------|------------|
| **摧毁 HQ 直胜** | 敌方主堡 HP 归零 → 所有敌方格子瞬间变为己方颜色（0.5s 波浪扩散特效） → 直接胜利。 |
| **时间耗尽比格数** | 3:00 倒计时归零 → 比较双方占领格子数量 → 多者胜。 |

### 经济

| Term | Definition |
|------|------------|
| **开地块** | 玩家点击己方可开格子 → 支付该格子的价格档位金币 → 走二次概率表得到建筑结果。**这是 v2 的核心玩家动作**。 |
| **HQ 产金** | 主堡被动产出金币，是开局的唯一收入源。 |
| **金矿产金** | 金矿被动产出金币，是中期经济引擎。 |

### AI

| Term | Definition |
|------|------------|
| **AI 难度档** | 简单 / 普通 / 困难三档。MVP 实现普通 + 困难。 |
| **对等 AI** | AI 与玩家遵守完全相同的经济规则（开地块付同样的钱、按同样的概率表出建筑），不作弊。 |
| **评分-选择算法** | AI 每 1.5 秒决策循环：枚举合法动作 → 启发式打分 → 按难度档选择动作（困难选最优，普通从 top-3 随机选）。 |
| **决策质量** | 区分难度档的核心参数：困难永远选最高分动作，普通有概率选次优动作模拟人类考虑不周。 |

### 交付

| Term | Definition |
|------|------------|
| **MuleRun** | 零依赖单文件 HTML 交付格式，所有 CSS/JS/资源内联在 `index.html`，浏览器直接打开即可运行。 |

## Decisions (stable, high-level)

- 平台：MuleRun 单文件 HTML
- 对战模式：单人对 AI（PvE）
- 截止日：2026-06-22
- 版本策略：v2 完全取代 v1（不保留 v1 的 reveal 翻格子机制和三角克制）
- 玩家动作：只有"开地块"一种核心动作（点击可开格子 → 扣费 → 随机出建筑）
- AI 难度：简单/普通/困难三档，MVP 做普通+困难，完全对等经济
