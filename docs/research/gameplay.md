# 占城大师 玩法调研

## 游戏概览

- **游戏类型：** 六边形领地争夺（1v1对战）
- **对战舞台：** 六边形棋盘
- **核心玩法：** 建造建筑 → 产出兵力 → 自动推进 → 占领敌方领地
- **上手门槛：** 极低，几分钟引导即可上手
- **对局时长：** 短平快（约3-5分钟）
- **平台：** 移动端/微信小游戏

## 核心玩法

### 1. 地图与开局

- 双方各占据地图两侧领地
- 各自拥有一座初始主城与少量金币
- **主城持续稳定产出金币**，是最基础的经济来源
- 地图地块需要逐步解锁/占领

### 2. 三大核心建筑

| 建筑 | 功能 | 策略意义 |
|------|------|----------|
| **兵营** | 不断输出各类士兵，士兵自动进攻 | 扩张主力，踏入敌方领地即可占据对应区域 |
| **金矿** | 产出更多金币，快速累计资源 | 加速经济滚雪球，为后续扩张打基础 |
| **防御塔** | 自动攻击范围内敌军 | 保护阵地安全，构建防线 |

### 3. 全自动作战

- 玩家不需随时监控兵种走位
- 不用关注地形影响
- 只需合理分配资源、规划建筑布局
- 依靠领地扩张和兵力压制，推掉敌方建筑或占据大部分版图即可获胜

## 核心策略

### 资源分配（核心策略点）

金币产出需要时间，资源有限时必须权衡分配：

1. **优先获取金矿** → 后续战力组建积攒更多资源
2. **搭建兵营** → 先一步输出兵力，抢占攻势主动权
3. **修筑防御工事** → 先稳固防线

### 见招拆招

双方都能看到对方的建筑排布，据此做出即时应对：

| 对方策略 | 应对方式 |
|----------|----------|
| 侧重经济发育 | 加快造兵，打乱对方节奏 |
| 集中搭建兵营 | 正面铺设防御塔，构建坚固防线 |
| 严密防御网 | 借助棋盘地块分布，迂回扩张，蚕食外围领地 |

## 兵种与品级系统

### 兵营品级（四大品级）

| 品级 | 特点 |
|------|------|
| 普通 | 基础兵种，产出快 |
| 稀有 | 战力较强 |
| 史诗 | 战力强，产出慢 |
| 传奇 | 最强兵种，产出最慢 |

- 投入不同数量金币建造兵营，对应产出高阶兵营的概率不同
- 投入资源越多，获取高品级兵营、产出强力兵种的几率越高

### 卡牌四大类型

- 人类、兽人、亡灵、机械
- 品级越高，士兵战力越强，但兵营建造概率越低，产出冷却时间越长

## 随机机制

### 神秘地块（问号格）

- 解锁门槛更低
- 建造结果充满不确定性
- 更大概率获得史诗和传奇兵营，也可能什么都没有
- 战局焦灼时"赌狗"心态强烈
- 解锁传奇兵种会触发全屏弹窗特效

## 养成模块（MVP范围外）

### 卡牌收集
- 推进游戏进程或付费购买获得新卡牌
- 胜利获得卡牌碎片，升级卡牌等级
- 每次对局前从卡牌库选择搭配出场阵容

### 开箱系统
- 胜利获得宝箱奖励
- 4个开箱槽位，开启需消耗时长
- 可看广告或用钻石加速

### 领地模块
- 使用木材资源扩张更多领地地块
- 每个地块掀开解锁不同建筑（随机）
- 提升战斗中主城攻击、生命值、金币产出速度等基础属性

## 关键体验设计

### 爽感设计
- 度过早期发育后，大量金矿→经济滚雪球
- 大量兵力单位集体冲锋，突破敌方防线
- 领地版图随着兵团推进不断向外拓展
- 结算动画完整展现棋盘尽数被己方占据的画面
- 色彩反转带来视觉冲击，放大成就感

### 节目效果
- 随机机制产生很强的节目效果
- 适合短视频和直播传播
- 传奇兵种解锁的全屏弹窗特效拉高玩家情绪

## 对 MVP 的启示

### 必须保留（核心循环）
- ✅ 六边形地图对战
- ✅ 兵营/金矿/防御塔三大建筑
- ✅ 全自动作战（简化操作）
- ✅ 资源分配策略
- ✅ 领地占领与视觉反馈

### 可简化/延后
- 🟡 卡牌收集/升级系统（长期养成）
- ❌ 多兵种品级（普通/稀有/史诗/传奇）
- ❌ 神秘地块概率玩法
- ❌ 开箱/广告商业化
- ❌ 领地经营养成模块

### MVP 设计取舍
- **兵种：** 3 兵种 + 克制链（步>骑>射>步），放弃品级系统
- **AI：** 固定行为树，放弃概率/策略复杂度
- **地图：** 固定种子生成，放弃随机问号格
- **交付：** 零依赖单文件 HTML，放弃后端/多人对战
